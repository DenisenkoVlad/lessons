Урок з каналу 2 (синтаксис)

Варіант 1 - Вірно, але слід поставити ";" після інструкції для надійності
Варіант 2 - Вірно (буде працювати), але слід поставити ";" після кожної інструкції для надійності
Варіант 3 - Не вірно, потрібно розділити інструкції ";", або починати кожну з нового рядка, а краще і те, і інше;)
Варіант 4 - Не вірно, не закрита лапка в першій інструкції, через це не буде працювати і друга

Урок з каналу 3 (var/let/const)

Варіант 1 - Не вірно, потрібно між двома об'явами змінних поставити ";" і для читабельності починати з нового рядка, або записати ось так: let user = 'Вася', age = 36;
Варіант 2 - Не вірно, не використано правило для написання lowerCamelCase - потрібно так: let myAge = 36;
Варіант 3 - Не вірно, стиль написання імен констант у верхньому регістрі використовують, якщо значення відоме заздалегідь, необхідно так: const blockSize = 14 + 50;
Варіант 4 - Вірно
Варіант 5 - Вірно, виводити значення з перемінної, об'явленої за допомогою var в блоці (на відміну від let) можна поза блоком (окрім function)

Урок з каналу 4 (типи даних)

Варіант 1 - Не вірно, для цієї задачі потрібно використати обернені лапки
Варіант 2 - Не вірно, повернеться Infinity
Варіант 3 - Не вірно, повернеться Undefined
Варіант 4 - Вірно, оператор ділення переводить тип даних в number
